# Constants
## Opacity
 - 62%

# Description
 - A black (#000000) overlay with [this opacity](#opacity)
 - Covers the entire screen
 - Is layered below the [swipe-up menu](./swipe_up_menu/swipe_up_menu.md) but above the [Globe viewport](./GlobeViewport.md) and [viewport2d](./viewport2d.md)
 - Blocks user input from reaching all the elements that it is layered above
