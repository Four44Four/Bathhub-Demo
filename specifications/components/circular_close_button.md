# Constants
## Fill color
 - #E06C89
## Image size pixels
 - 18
## Padding size pixels
 - 15

# Properties
## X position
 - Type:
    - Integer
## Y position
 - Type:
    - Integer
## On click callback
 - Type:
     - Function that takes in a click event

# Description
 - Is a [viewport2d button](./viewport2d_button.md) with the following properties:
     - [X position](./viewport2d_button.md#x-position) of [x position property](#x-position)
     - [Y position](./viewport2d_button.md#y-position) of [y position property](#y-position)
     - [Image](./viewport2d_button.md#image) is the [X symbol icon](../resources.md#x-symbol-icon) with color white (#000000)
     - [Image size](./viewport2d_button.md#image-size) is [this size](#image-size-pixels)
     - [Padding](./viewport2d_button.md#padding) is [this size](#padding-size-pixels)
     - [Fill color](./viewport2d_button.md#fill-color) is the [negative color](#fill-color)
     - Is [circular](./viewport2d_button.md#circular-flag)
     - [On click callback](./viewport2d_button.md#on-click-callback) is the [on click callback property](#on-click-callback)
     - [Hover interaction behavior](./viewport2d_button.md#hover-interact-behavior) is "darken"
     - Has a [outline thickness](./viewport2d_button.md#outline-thickness) of 0 (no outline)
