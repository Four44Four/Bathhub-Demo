# Image usage policy
 - Whenever an image (svg, ng, jpg, etc...) is to be used:
    - Pass it by creating an [image resource descriptor](./image.md) first
    - If there are any components that still rely on a file path system:
       - Create a warning about a spec misalignment but do not change it unless explicitly instructed to
