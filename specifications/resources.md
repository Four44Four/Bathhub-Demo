# Mono-color icon policy
 - All mono-color icons should be uniformly black
    - If an mono-color icon isn't exactly #000000:
       - Notify developer immediately
 - Whenever a mono-color icon is used:
    - If a color is not specified:
       - The icon will be displayed as black (#000000)
    - Use CSS filters to manipulate their color to the target color
